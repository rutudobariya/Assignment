Bootstrap Container

Containers are very essential and basic building blocks of bootstrap that wrap a page’s content.

Containers are defined within the container class (.container).

Basically, there are three types of container classes available in bootstrap:

  1. Default-Container(container)
  2. Responsive-Container(along with sm, md, lg, xl, xxl)
  3. Fluid-Container(container-fluid)

1. Default-Container The default container uses ‘.container’ class. It provides a responsive fixed-width container.(container occupies 80% width)


2. Responsive-Container (along with sm, md, lg, xl)
container-sm--Wide for small breakpoint

container-md--Wide for medium breakpoint

container-lg--Wide for large breakpoint

container-xl--Wide for extra large breakpoint

3. Fluid-Container(container-fluid) Fluid-container uses the ‘.container-fluid’ class. It is used for a full-width container, spanning the entire width of the viewport.
